# CV Builder Project

This project is a JavaFX application for creating CVs (Curriculum Vitae) easily and efficiently. It allows users to fill in their personal information and generate a formatted CV.

## Features
- User-friendly interface
- Supports multiple CV entries
- Preview of the final CV

## Requirements
- Java 11 or higher
- JavaFX SDK

## How to Run
1. Clone the repository
2. Navigate to the project directory
3. Run `mvn clean install` to build the project
4. Run the application using `java -jar target/yourapplication.jar`
