## Requirements
- **Java 21**
- **JavaFX 21**
- **Maven**

## Installation & Running Instructions
### Maven Method
1. Clone the repository:
   ```bash
   git clone https://github.com/AbirHasanArko/Arko_2207053_CVBuilder.git
   cd Arko_2207053_CVBuilder
   ```
2. Build the project:
   ```bash
   mvn clean install
   ```
3. Run the application:
   ```bash
   mvn javafx:run
   ```

### IntelliJ Method
1. Open IntelliJ IDEA.
2. Click on `Open` and select the cloned project folder.
3. Set the JDK to **Java 21**.
4. Build the project and run the main class to start the application.

## Project Structure
- `src/main/java`: Contains Java source files
- `src/main/resources`: Contains resource files like FXML
- `src/test`: Contains test files

## Technologies Used
- **JavaFX** for GUI
- **Maven** for project management
- **JUnit** for unit testing

## How to Use
### Step-by-Step Guide
1. Launch the application.
2. Navigate through different screens:
   - **Home Screen**: Get an introduction and start creating your CV.
   - **Form Screen**: Fill in your personal details, education, and work experience.
   - **Preview Screen**: Review your entered data and see the final CV layout.
3. Export your CV once completed.
